\subsubsection{Detailed Description}
A connection handle identifies a particular connection. Handles are used by the connection registry to return enumeration results and to instantiate connections.

When using \doxyref{Connection\+Handle}{p.}{d6/d37/classlime_1_1ConnectionHandle} to specify an enumeration hint, simply fill in only the fields that the user is interested in. Each default field (empty string, -\/1 index) will be ignored by the connection registry entry discovery implementation.

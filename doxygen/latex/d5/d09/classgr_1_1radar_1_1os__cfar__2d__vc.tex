This block estimates peaks of a given matrix. A matrix can be represented as a combination of vectors and tagged streams. The algorithm uses around the cell under test (C\+UT) on each side samp\+\_\+compare samples to estimate the noise floor. samp\+\_\+protect is a protected are around the C\+UT which is not used for acquiring compare samples. If the magnitude square of the C\+UT is greater than the threshold the matrix item is accepted.

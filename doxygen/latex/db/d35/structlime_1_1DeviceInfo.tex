\subsubsection{Detailed Description}
Information about the set of available hardware on a device. This includes available I\+Cs, streamers, and version info.

This structure provides S\+PI slave addresses for one or more R\+F\+I\+Cs and slave addresses or I2C addresses for commonly supported I\+Cs. A -\/1 for an address number indicates that it is not available.

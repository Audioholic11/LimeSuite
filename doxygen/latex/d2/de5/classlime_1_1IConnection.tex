\subsubsection{Detailed Description}
\doxyref{I\+Connection}{p.}{d2/de5/classlime_1_1IConnection} is the interface class for a device with 1 or more Lime R\+F\+I\+Cs. The \doxyref{L\+M\+S7002M}{p.}{db/d1d/classlime_1_1LMS7002M} driver class calls into \doxyref{I\+Connection}{p.}{d2/de5/classlime_1_1IConnection} to interface with the hardware to implement high level functions on top of low-\/level S\+PI and G\+P\+IO. Device developers will implement a custom \doxyref{I\+Connection}{p.}{d2/de5/classlime_1_1IConnection} for their hardware as an abstraction for streaming and low-\/level S\+PI and configuration access.

\subsubsection{Detailed Description}
Create an overloaded instance of a \doxyref{Connection\+Registry\+Entry}{p.}{d2/d45/classlime_1_1ConnectionRegistryEntry} to register discovery and factory functions into the system. The \doxyref{Connection\+Registry\+Entry}{p.}{d2/d45/classlime_1_1ConnectionRegistryEntry} should be created prior to discovering and instantiating connections. One recommended use is at static initialization time.

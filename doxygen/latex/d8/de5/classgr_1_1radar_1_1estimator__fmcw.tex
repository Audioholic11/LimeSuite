Input messages are data with the identifier \textquotesingle{}frequency\textquotesingle{} of the up-\/chirp, down-\/chirp and CW part. If data is available on all three message ports the estimation starts. The velocity is estimated with the frequency information of the CW block and the range is estimated with the up and down chirp.

#define LIME_SUITE_API_VERSION   0x20180600

API version number which can be used as a preprocessor check. The format of the version number is encoded as follows: (major << 16) | (minor << 8) | (8 bit increment). Where the increment can be used to indicate implementation changes, fixes, or API additions within a minor release series.

The macro is typically used in an application as follows:

1 #if defined(LIME_SUITE_API_VERSION) && (LIME_SUITE_API_VERSION >= 0x20161200)
2 // Use a newer feature from the LimeSuite library API
3 #endif

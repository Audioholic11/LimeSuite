overridevirtual

Bulk SPI write/read transaction.

The transactSPI function is capable of bulk writes and bulk reads of SPI registers in an arbitrary IC (up to 32-bits per transaction).

The readData parameter may be NULL to indicate a write-only operation, the underlying implementation may be able to optimize out the readback.

Parameters
addrthe SPI device address
writeDataSPI bits to write out
[out]readDatastores readback data
sizethe number of SPI transactions
Returns
the transaction success state

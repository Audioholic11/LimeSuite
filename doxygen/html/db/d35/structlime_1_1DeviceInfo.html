Detailed Description

Information about the set of available hardware on a device. This includes available ICs, streamers, and version info.

This structure provides SPI slave addresses for one or more RFICs and slave addresses or I2C addresses for commonly supported ICs. A -1 for an address number indicates that it is not available.
